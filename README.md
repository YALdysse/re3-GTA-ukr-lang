# [Ukr] Українська локалізація для проєкту re3 (reverse engineered GTA III)

Мета цього репозиторію: адаптувати українську локалізацію від Andrulko, котра працює тільки на Windows-версії гри, для роботи з вільної реалізацією GTA 3 - re3. Таким чимом, ця локалізація буде успішно працювати на Linux, MacOS та Windows версіях гри.

### Відомості про локалізацію

За основу було взято український gxt з "Тройного русификатора" від DimZet. Автором цієї локалізації, здається (скоріш за все, бо інших український локалізацій я не знаходив), є Andrulko.

Коротко:

+ Адаптовано для використання з портом re3 для Linux, MacOS та Windows;
+ Шрифт високої роздільної здатності;
+ Назви автомобілів, персонажів та районів записано транслітом

### Інсталяція

Увага: поточна версія локалізації встановлюється замість російської локалізації!

1. З розділу з [релізами](https://github.com/YALdysse/re3-GTA-ukr-lang/releases) завантажити архів з gxt та розпакувати його. Файл `ukrainian.gxt` перейменувати в `russian.gxt` та скопіювати з заміною в каталог з грою `./text`.
1. Також скопіювати файл шрифту `./assets/models/fonts_r.txd` з репозиторію в каталог з грою `./models` з заміною файлу.
1. Запустити re3 та в налаштуваннях мови обрати `Русский` (назва не змінювалась з метою сумісності).
1. Насолоджуватись солов'їною!

### Структура каталогів для розробників

+ `decompilation` - все, що необхідно для декомпіляції gxt, котра була створена для оригінальної GTA III 
+ `compilation` - все, що потрібно для компіляції виправленого txt в gxt для re3
